jQuery Singular.js

SingularJSis a jQuery plugin to create SinglePage Website.

How to use

HTML Markup is too easily.

<div class="singular">
<section id="sec01" class="singular-section"></section>
<section id="sec02" class="singular-section"></section>
<section id="sec03" class="singular-section"></section>
<section id="sec04" class="singular-section"></section>
</div>

Options

[text] section
Section's classname.
(Default: '.singular-section')
[text] nav
Nav's classname.
(Default: '.singular-nav')
[text] prev
The classname that links to the previous section.
(Defualt: '.singular-prev')
[text] next
The classname that links to the next section.
(Defualt: '.singular-next')
[text] navActiveClass
The classname current section
(Default: singular-active)
[int] scrollSpeed
scroll speed(ms)
(Default: 1000)
[text] easing
Easing
(Default: 'swing')
[bool] mousewheel
Scroll using the mouse wheel
(Default: false)
